## Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/your-username/SehTajDesk.git
    ```
2. Navigate to the project directory:
    ```bash
    cd SehTajDesk
    ```

## Usage

1. Install dependencies:
    ```bash
        pip install -r requirements.txt
    ```
2. Run
    ```
    python3 main.py
    or py main.py
    ```
